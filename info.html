HTML


HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web.
Define el significado y la estructura del contenido web.
Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

Un elemento HTML se distingue de otro texto en un documento mediante "etiquetas", que consisten en el nombre del elemento rodeado por "<" y ">" .

Pero, HTML no trabaja solo (en realidad es solo el esqueleto de nuestra pagina web).
Tambien debemos usar CSS (que se encarga de los estilos) y JS (que se encarga de la interactividad)

Ahora vamos con lo que nos importa... las etiquetas!

Lo primero que debemos conocer son las etiquetas head y body

La etiqueta head va a contener toda la informacion de configuración, título e ícono de nuestra página

Por otro lado, la etiqueta body va a contener nuestra página en general, todo aquello que el usuario va a poder ver y con lo que podra interactuar.

Dentro del body vamos a contener muchas etiquetas que nos van a ayudar a darle forma al sitio web. Por ejemplo:

Etiquetas header, main y footer

Como su nombre lo indica cada etiqueta contendra una seccion diferente de nuestro sitio

Para crear un título lo hacemos con la etiqueta:

h1

Conforme el número sea mayor el tamaño de la fuente y la importancia del titulo sera menor.

Es una buena practica tener solo un h1 por página

Para crear un párrafo lo hacemos con la etiqueta:

p

Para crear un formulario lo hacemos con la etiqueta:

Un formulario me sirve para que el usuario pueda cargar informacion.

Dentro del formulario se utilizan los inputs y botones, ademas se pueden agregar titulos o parrafos.
